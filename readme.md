# Emotion
a rouge-like card game


## 4.24
 - 基本框架, 可以在Inspector面板中进行一场完整的游戏
 - 基本对局流程,包括抽牌,选牌,打出,造成效果,游戏状态判断与切换
 - 基本的人物、敌人属性
 - 基本卡牌属性,两种卡牌“怒气”、“治愈”
 - 基本卡牌效果,造成伤害和回血
 - 简单的敌人AI:随机打出一张牌
 - 操作方式:AD选择牌,K打出,打出后当前手牌置空,需要再次选择,J结束自己的回合,AI随机出牌,然后又是自己的回合
